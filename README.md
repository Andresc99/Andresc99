## 📂 Proyectos Destacados

### 🏦 [Churn de Clientes Bancarios — Power BI](https://github.com/Andresc99/Churn-de-Clientes-Bancarios---PowerBI)

> Análisis predictivo de abandono de clientes con dashboard interactivo en 3 niveles.

- Pipeline ETL sobre +10,000 registros de clientes
- Alemania encabezó el abandono con **32.4%** vs 16% en Francia y España
- Las clientas mostraron un **25% de churn** frente al 16.4% masculino
- Dashboard con drill-down: tendencias macro → segmentos individuales
